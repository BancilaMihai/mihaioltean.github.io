Traceless Genetic Programming

Traceless Genetic Programming (TGP) is a Genetic Programming variant as it evolves a population of computer programs.

The main difference between the TGP and GP is that TGP does not explicitly store the evolved computer programs.

TGP is useful when the trace (the way in which the results are obtained) between the input and output is not important. In this way the space used by traditional techniques for storing the entire computer programs (or mathematical expressions in the simple case of symbolic regression) is saved.
